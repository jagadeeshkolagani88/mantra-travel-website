# 🌍 Travel Explorer Website

## 📌 Project Overview
Travel Explorer is a responsive travel website built using HTML, CSS, and JavaScript. It helps users explore popular destinations, view travel packages, browse a photo gallery, and contact the travel agency.

---

## ✨ Features
- Responsive design
- Home page with hero section
- About Us page
- Popular Destinations page
- Travel Packages page
- Image Gallery
- Contact Form
- Mobile-friendly layout
- Interactive buttons and form validation using JavaScript

---

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
